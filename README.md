# Simulação — Interação da Radioatividade com a Matéria

Mini-site para apresentação.

## Como abrir
Abra o arquivo `index.html` no navegador.

## O que tem
- Simulador de radiação alfa, beta e gama atravessando blindagens.
- Controle de material e espessura.
- Medidor de dose transmitida.
- Simulador visual de contador Geiger.
- Representação da penetração no corpo humano.
- Gancho final para perícia nuclear e ciência forense.
